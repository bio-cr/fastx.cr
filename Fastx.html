    def self.encode_bases(sequence : IO::Memory | String, *, iupac : Bool = false) : Slice(UInt8) #

    Converts a DNA sequence (String or IO::Memory) to a UInt8 slice, where each base is encoded as a single byte. When iupac is true, supports IUPAC nucleotide codes (R, Y, S, W, K, M, B, D, H, V). When iupac is false, only standard bases (A, C, G, T, N) are preserved. Non-recognized characters are converted to N (78u8). This representation is suitable for byte-wise or array processing.
